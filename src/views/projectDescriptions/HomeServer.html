Home Server

A cloud service replacement that is meant to run a variety of services, from VPN, mass file storage, AI training, media streaming, VM services, and more without relying on any 3rd party providers or subscriptions.

Project Overview

This project was created to avoid utilizing 3rd party subscription based cloud services and have total control over my own data. The main goal was to create a home server that could run a variety of services such as VPN, mass file storage, AI training or agents, media streaming, VM services, home automation services and more. This server was built using readily available hardware components and open-source software to keep costs low and ensure flexibility.

While currently the services are fairly basic, the server is scalable and will be expanded upon in the future and used in new projects I have planned. At it's core it runs a Proxmox hypervisor to manage virtual machines and containers, and uses ZFS for mass file storages.
